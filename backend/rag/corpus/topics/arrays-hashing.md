# Arrays and Hashing

Arrays store elements in contiguous memory; hashing maps keys to values in average O(1) time.

**Patterns:** two-pass frequency count, sliding window with frequency map, prefix sums with hash map for subarray sum.

**Classic problems:** Two Sum (hash map complement), Contains Duplicate, Longest Consecutive Sequence.

**Prerequisites:** basic iteration, understanding of time vs space tradeoffs.

**Next topics:** two pointers on sorted arrays, stacks for parsing.
