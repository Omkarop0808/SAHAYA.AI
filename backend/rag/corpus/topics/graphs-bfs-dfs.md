# Graphs: BFS and DFS

Breadth-first search explores layer by layer using a queue; shortest path in unweighted graphs.

Depth-first search explores aggressively along edges before backtracking; cycle detection, topological sort foundations.

**Patterns:** grid as implicit graph, color states (white/gray/black), multi-source BFS.

**Complexity:** O(V + E) time, O(V) space for visited structures.
